Q.1 How many operators do we have in JS ?

ANS

        We have 6 types of operators in JS they are
             1. Arithmetic Operators : + - * / %
             2. Comparison Operators : == === != !==
             3. Logical Operators : && || !
             4. Assignment Operators : =
             5. Conditional Operators
             6. Ternary Operator ?:

Q.3 Difference between undefined vs not defined vs NaN

ANS

        
Undefined
            undefined is a global variable that JavaScript creates at run time. JavaScript assigns undefined

            to any variable that has been declared but not initialized. In other words, in a case where no

            value has been explicitly assigned to the variable, JavaScript calls it undefined.


        
null
            It is one of JavaScriptâ€™s primitive data type and is treated as falsy for boolean operations.null is

            an empty or non-existent value and null must be assigned. We use null when we want to

            explicitly declare that a variable is empty.


        
not defined
            A not defined is a variable that is not declared inside the code at a given point of time with

            declaration keyword like var, let, or const.

Q.4 What is callback hell

ANS

         
Callback Hell is a pattern with multiple nested callbacks which makes code hard to read.
         The callback hell looks like a pyramid structure.

        Ex-

        setTimeout(()=>{
            console.log("hellow1")
            setTimeout(()=>{
                console.log("hellow2")
                setTimeout(()=>{
                    console.log("hellow3")
                    setTimeout(()=>{
                        console.log("hellow4")
                        setTimeout(()=>{
                            console.log("hellow5")
                            
                        },1000)
                        
                    },1000)
                    
                },1000)
                
            },1000)
            
        },1000)
